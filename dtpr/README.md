# Benchmark

This open-source template helps project teams create a clear, public-facing DTPR transparency page without needing to design or build a website from scratch. Users can quickly adapt the experience by changing plain configuration values in `js/config.js`, including the project name, description, survey link, background image, colors, footer links, and DTPR cards; no build tools or coding workflow are required for basic customization. Run it locally with `py -m http.server 8000`, open `home.html` to view the public landing page, `index.html` to view the DTPR information cards, and `signage.html` to download editable signage assets. The goal is to make it easy for researchers, civic teams, and community-facing projects to reuse, publish, and maintain transparent communication materials.
